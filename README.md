# Simple-Blockchain-SnakeCoin
SnakeCoin is a simple Python-based blockchain prototype that demonstrates block creation, proof-of-work mining, and secure transaction linking. It stores each block with a timestamp, data, proof, and previous hash, showcasing core blockchain concepts in an easy-to-understand implementation.

# Minimal Python Blockchain

A minimal blockchain implementation in Python for educational purposes.

## Overview

This project demonstrates the basic structure of a blockchain, following a simplified model to explain the concepts of blocks, chains, and hashing.
